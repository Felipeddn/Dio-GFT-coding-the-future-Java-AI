Um código java pode ser escrito em um editor de texto, porém usar as IDEs traz um ganho de produtividade, esses são os beneficios proporcionado pelas IDEs:

Acelera a escrito com Auto-complite ou Intellisense

formatação de palavras e bloco de códigos

Análise de erro de sintaxe

Compilação de programas

Depuração (acompanhamento) de execução do programa.

AS IDEs mais utilizadas do mercado

Eclipse 
Comum em ambientes corporativos pelo design simples e para trabalhar com Java Swing.

IntelliJ IDEA
Uma IDE incrivel porém possui componentes pago. 

Visual studio code
Desenvolvido pela Microsoft, também muito utilizado para outras linguagem.

Apache NetBeans IDE
