Configurando o ambiente
O Java possui o JDK (Java Development Kit - pacote de desenvolvimento Java) e a JRE (Java Runtime Enviroment - Ambiente de execução Java) 

Para desenvolver em Java precisamos do JDK instalado. Esse JDK tem uma versão é importante saber escolher a versão da IDE.

Instalar o JDK manualmente exige certa compreensão de variáveis de ambiente, no primeiro momento o facilitador deixou claro que não iremos nos preocupar com isso, pois as IDEs irá nos facilitar em relação a isso.
